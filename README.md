# Drip Social

I created this project as a part of my web development journey.

## Tech i used :

- Next js (app router)
- Typescript
- Tailwind Css
- Firebase (for database)
- Next-auth (for authentication)
- Cloudinary (for images management)




## Basic config before you started this project

```
npm install 
```
```
npm run dev

http://localhost:3000
```
